git add --all =  To add every change to staging area
git reset rewinds commits but keeps your work.
 Moves HEAD to the specified commit
 Resets the staging area (index)
 Does NOT change your working directory

git reset --hard rewinds commits and deletes your work.
git add .  = To add change withing the current directory 
git add * - stages only modified and new file added
